SPECIFICATIONS
7 Themed Environments
Forest x4 (seasonal), City x3 (geographic). Each biome features unique sprites across 5 contribution levels.
Ambient Animations
Deer wander through forests. Taxis drive in NYC. Cherry blossoms drift in spring. Snow falls in winter.
AI-Generated Sprites
Gemini API pipeline produces all 101 sprites for ~$3.76. Full validation suite ensures consistency across themes.
Extensible Theme System
JSON schema + prompt templates. Create entirely new biomes without touching rendering code.
3 Rendering Modes
Squares (original GitHub style), Cubes (isometric), or Both overlaid. Toggle freely from the popup.
Smart Page Detection
Handles GitHub's Turbo navigation and SPA routing. Works on any profile page, including your own.
INSTALLATION PROCEDURE — REV 1.0
01
Clone Repository
$ git clone https://github.com/jeanluciradukunda/contribution-lands.git
02
Build Extension
$ cd extension && pnpm install && pnpm build
03
Load in Chrome
Navigate to chrome://extensions Enable "Developer mode" toggle Click "Load unpacked" → select extension/dist/
04
Verify
Visit any GitHub profile The contribution graph transforms automatically
